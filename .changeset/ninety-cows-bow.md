- `any.entriesof` - use `any.entriesOf` instead
- `any.entryof` - use `any.entryOf` instead
- `any.keysof` - use `any.keysOf` instead
- `any.subtypeof` - use `any.subtypeOf` instead
- `any.indexedby` - use `any.indexedBy` instead
